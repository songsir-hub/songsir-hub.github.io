    引子：一场持续了五千年的日出

    想象一下：你站在一块青灰色的石岗上，脚下是夯实的土台，面前是广袤的平原。东方的天际线渐次发白，然后——一轮红日恰好从两座山峰之间的豁口跃出，第一道阳光直直地打在你手中的大口尊上。

  • 在 Windows 上部署 FastGithub 踩坑记录

    2026-07-09#软件技巧

    背景

    受够了 GitHub 相关站点的访问稳定性长期不稳定，也试过修改 host 文件等办法，麻烦又效果差，最终下决心试试 FastGithub。由于 GitHub 站点基本上不去，原版 dotnetcore/FastGithub 仓库已不可用、Gitee 镜像只提供源码而无现成二进制，在 AI 建议下，最终采用仍在维护的 creazyboyone/FastGithub v2.1.5——其工作机制与原版一致。本文记录在本机及第二台电脑上部署、排障、并沉淀可复用资产的全过程。

    一、获取与解包

    通过 ghproxy.net 这类镜像拉取 fastgithub-win-x64.zip，避免直接连接 GitHub。下载后校验 SHA256，解包得到自包含的 app\ 目录：其中的 fastgithub.exe 自带 .NET 运行时，无需另装环境。部署目录最终统一为 D:\tools\FastGithub，所有电脑共用同一份。

    二、运行机制的要点

    FastGithub v2.1.5 在 Windows 上不依赖系统代理，而是用 WinDivert 驱动做 DNS 透明拦截：把 github.com 等域名的解析结果改写为 127.0.0.1，再由本地 MITM 代理接管 TLS 并回源。这一点决定了后续的排障方向——系统代理应当为空，证书必须被系统信任。

    2026-07-06#东夷莱夷

    在山东半岛中部，流着一条谜一样的河。

    它全长不过130公里，放在中国众多江河中毫不起眼。但它身上却挂着好几个让人挠头的问题：为什么一条修建于700年前的运河南段，至今仍被沿岸老人叫作"运粮河"？明明如今它汇入大沽河后方才入海，但历代文献从不将它视为大沽河的支流——古代它到底怎么入海的？更奇怪的是，这样一条曾经扛起大元帝国60%漕运量的生命线，为何在鼎盛之后两度废弃，最终被清雍正帝一纸"永不疏浚"打入冷宫？还有，2000年代提出的"胶莱人工海河"构想，号称中国版苏伊士运河，为何至今只见图纸不见开工？

  • 莱国双源假说：东莱与西莱的文化分野

    2026-06-15#东夷莱夷

    引言

    “莱"字在先秦文献中并不罕见，但其所指并非单一。胶东半岛的莱国（东莱）与今山东西部的莱夷（西莱），在地理上相距遥远，文化背景也存在显著差异。长期以来，学界多将二者混同讨论，或简单地以"莱夷"统称之。

    本文尝试提出一个"双源假说”：东莱与西莱的命名与文化渊源可能分属两条截然不同的脉络。

    东莱：草本植物的命名

    “莱"字的本义，《说文》释为"藜也”——一种野生草本植物，即今日常见的灰藜（Chenopodium album）。这种植物耐盐碱、适应性强，在胶东沿海地区广泛分布，历史上曾是贫民的重要食物来源。

    胶东半岛的"莱"地名，很可能直接源于这一植物的分布特征。类似的命名逻辑在上古地名中并不少见——以当地最具辨识度的自然特征命名聚落或地域。
